## 📝 Kesimpulan Pembelajaran (Tugas 4, 5, & 6)

### 1. Struktur Data Python (Tugas 4)
Pada modul ini, saya mendalami fleksibilitas **List**, kekakuan **Tuple** yang bersifat *immutable*, serta keunikan **Set** dalam menangani data duplikat. Saya juga mempraktikkan penggunaan **Dictionary** untuk pemetaan data berbasis *key-value*. Salah satu teknik favorit saya adalah *List Comprehension*, yang memungkinkan transformasi data menjadi jauh lebih ringkas dan efisien tanpa memerlukan banyak baris kode (Pythonic Way).

### 2. Fungsi & Pemrograman Berorientasi Objek (Tugas 5)
Fokus utama tugas ini adalah pengorganisasian kode agar lebih modular. Saya belajar bagaimana membangun fungsi yang dapat digunakan kembali (*reusable*) dan menerapkan konsep **Object-Oriented Programming (OOP)** melalui pembuatan *Class Student*. Dengan mendefinisikan atribut dan *method* khusus (seperti `__str__`), struktur kode menjadi lebih logis, mudah dibaca, dan mempermudah pengelolaan data entitas.

### 3. Analisis Data & Modul Eksternal (Tugas 6)
Tugas penutup ini memperkenalkan "kekuatan utama" dalam pengembangan AI, yaitu library **NumPy** untuk perhitungan statistik cepat dan **Pandas** untuk manajemen data dalam bentuk tabel (*DataFrame*). Selain analisis data, saya juga mengintegrasikan sistem **File I/O** untuk melakukan ekspor hasil ringkasan analisis secara otomatis ke dalam format `.txt`, yang sangat berguna untuk dokumentasi hasil eksperimen secara permanen.
